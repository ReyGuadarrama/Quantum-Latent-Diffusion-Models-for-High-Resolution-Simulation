This repository/folder presents the preliminary code and implementation attempts for the VAE. An Autoencoder was developed first, followed by the VAE architecture, which was evaluated across three distinct datasets: 
- MNIST
- Cats vs. Dogs
- Jet Dataset

Variational Autoencoders (VAEs) are a class of generative models in machine learning designed to synthesize new data by learning the underlying distribution of their training inputs. Furthermore, they are capable of executing traditional autoencoder tasks, including noise reduction.
